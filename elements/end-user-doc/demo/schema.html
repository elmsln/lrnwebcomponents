end-user-doc Page

The contents property is set by using a javascript object with the following properties:

  • id (string): recommended, a unique identfier that serves as anchor
  • title (string): recommended, page heading
  • contents (array): required, array of page content

end-user-doc Sections

A section or subsection can be added to a page's or section's contents array by using a javascript object with the following properties:

  • id (string): recommended, a unique identfier that serves as anchor
  • title (string): recommended, section heading
  • contents (array): required, array of section content

end-user-doc Images

You can add images to any contents array by creating an object with the following properties:

  • id (string): recommended, a unique identfier that serves as anchor
  • alt (string): recommended, image alt text for accessibility
  • src (string): required, image source
  • srcset (string): optional, image source set for responsive images
  • sizes (string): optional, image sizes for responsive images
  • caption (string): optional, caption if image will be a figure

end-user-doc Cheatsheets

You can add cheatsheet reference tables to a contents array by creating an object with the following properties:

  • id (string): recommended, a unique identfier that serves as anchor
  • title (string): recommended, table caption
  • cheatsheet (object): required, object with the following properties:
    • columns (array): recommended, array of column headers
    • rows (array): required, nested array of rows, where each row array contains a string for each column

end-user-doc Step-by-Step Instructions

You can add step-by-step lists to a contents array by creating an object with the following properties:

  • id (string): recommended, a unique identfier that serves as anchor
  • listStyle (string): optional, space-separated list styles in order of least nested to most nested
  • steps (array): required, array of steps, which can include nested arrays of substeps
