end-user-doc Default Display

By default, section objects with the toc property will display with a list of buttopns to navigate to each content item. Clicking the buttons will show the rendered sections.

end-user-doc Displayed as a Single Page

If display-mode is set to all, all documentation sections will be rendered on a single page.

end-user-doc Automatic Display

If display-mode is set to a number, section objects with items that number greater than or equal to the display-mode number, will display with a list of buttons to navigate to each content item, regardless of whether the sections have the toc property.

end-user-doc Navigate to Section

You can navigate to a specific section by setting the current-section.

end-user-doc Hide Breadcrumbs

If you do not want to display breadcrumbs, you can set hide-breadcrumbs.
